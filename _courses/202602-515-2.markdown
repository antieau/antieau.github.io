# Expected topics 
- presentable $\infty$-categories
- compactly assembled $\infty$-categories
- dualizable stable $\infty$-categories
- $\infty$-categories of sheaves on topological spaces
- $6$-functors for sheaves on topological spaces
- continuous $K$-theory of sheaves on topological spaces (Efimov's theorem)
